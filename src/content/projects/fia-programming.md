## Overview

Programming and technology program at FIA (Fundação Instituto de Administração), with a strong emphasis on practical application — writing real code that solves real problems.

## Focus Areas

- **Web Development** — Front-end and back-end fundamentals, REST APIs
- **Scripting & Automation** — Python for data processing and tooling
- **Database Access** — SQL queries, ORM patterns, data modeling
- **Software Engineering** — Version control, testing, code review practices

## Philosophy

The program pushed for understanding *why* code works, not just *how* to write it. That foundation shapes how I approach every problem — understand the data model first, then write the code.

## Visão Geral

Programa de programação e tecnologia na FIA (Fundação Instituto de Administração), com forte ênfase na aplicação prática — escrever código real que resolve problemas reais.

## Áreas de Foco

- **Desenvolvimento Web** — Fundamentos de front-end e back-end, APIs REST
- **Scripts & Automação** — Python para processamento de dados e ferramentas
- **Acesso a Banco de Dados** — Consultas SQL, padrões ORM, modelagem de dados
- **Engenharia de Software** — Controle de versão, testes, práticas de revisão de código

## Filosofia

O programa exigia entender o *porquê* do código funcionar, não apenas o *como* escrevê-lo. Essa base molda como abordo cada problema — entender o modelo de dados primeiro, depois escrever o código.
